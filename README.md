# ⚽ Predição de Resultados na Premier League via Índice de Proatividade (IP)

Este repositório contém o código técnico e os dados utilizados para o Trabalho de Conclusão de Curso (TCC) do **MBA em Data Science e Analytics da USP/Esalq**.

## 📊 Resumo do Projeto
O objetivo deste estudo foi transcender as estatísticas tradicionais de futebol através da criação do **Índice de Proatividade (IP)**. O indicador foi validado utilizando:
* **Clusterização (K-Means):** Para identificar taxonomias táticas.
* **Random Forest:** Para prever vitórias com uma acurácia de **64,89%**.

## 🛠️ Tecnologias Utilizadas
* Python 3.x
* Pandas & Numpy (Processamento)
* Scikit-Learn (Modelagem)
* Matplotlib & Seaborn (Visualização)

## 📁 Estrutura do Repositório
* `epl_final.csv`: Base de dados consolidada.
* `TCC_Premier_League.ipynb`: Script completo com as análises e modelos.
